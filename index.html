    - messagebox es un submódulo de tkinter que permite mostrar mensajes emergentes (como alertas o información).

    - Crea un botón para cada celda del tablero.

    def click(self, INTrow, INTcolumn):

    - Define una función que se ejecuta cuando se hace clic en un botón del tablero.

    if self.buttons[row][column]['text'] == "":
    - Verifica si el botón en la posición row, column está vacío. Si el botón ya tiene un texto (o "x" o "o"), no se hará nada.

    self.buttons[row][column]['text'] = self.player_turn
    - Si el botón está vacío, se le asigna el texto correspondiente al turno del jugador actual ("o" o "x").

    if self.check_win():
    - Llama a la función check_win() para verificar si algún jugador ha ganado después de hacer clic.

    messagebox.showinfo("Juego terminado", f"El jugador {self.player_turn} gana.")
    - Si hay un ganador, muestra un mensaje emergente con la información de quién ha ganado.

    self.window.quit()
    - Después de mostrar el mensaje de ganador, cierra la ventana y termina el juego.

    self.player_turn = "x" if self.player_turn == "o" else "o"
    - Alterna el turno entre los jugadores. Si el turno era de "o", ahora será de "x", y viceversa.

    def check_win(self):

    - Define una función para verificar si hay un ganador.

    for row in self.buttons:
    - Recorre cada fila de la matriz self.buttons.

    if row[0]['text'] == row[1]['text'] == row[2]['text'] != "":
    - Verifica si los tres botones de la fila tienen el mismo texto y no están vacíos. Si es así, significa que hay un ganador en esa fila.

    for column in range(3):
    - Recorre las tres columnas del tablero.

    if self.buttons[0][column]['text'] == self.buttons[1][column]['text'] == self.buttons[2][column]['text'] != "":
    - Verifica si los tres botones de la columna tienen el mismo texto y no están vacíos. Si es así, hay un ganador en esa columna.

    Diagonales:
    - Se comprueban las dos diagonales del tablero:
    - Diagonal principal: self.buttons[0][0], self.buttons[1][1], self.buttons[2][2].
    - Diagonal secundaria: self.buttons[0][2], self.buttons[1][1], self.buttons[2][0].
